# PhotoManagerApp

a simple Photo Management App (Gallery) written in kotlin.

## Technologies used
  - Android Studio
  - Kotlin
  - XML
  - Picaso Library

## Demo

you can build the project in android studio, or u can install the apk on ur device.

  - minimum sdk version : 21
  - target sdk version : 32
